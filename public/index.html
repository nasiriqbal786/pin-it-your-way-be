Available Endpoints

GET Pinterest Auth Initialization

/.netlify/functions/pinterest-auth

Generates Pinterest OAuth authorization URL with secure state parameter.

GET OAuth Callback Handler

/.netlify/functions/pinterest-token?code=xxx&state=xxx

Handles Pinterest OAuth callback and exchanges authorization code for access token.

POST Pinterest API Proxy

/.netlify/functions/pinterest-token

Secure proxy for Pinterest API calls (boards, pins, user info).

Supported actions: getBoards, createPin

Environment Configuration

The following environment variables should be configured in your Netlify dashboard:

  • PINTEREST_APP_ID - Your Pinterest App ID
  • PINTEREST_APP_SECRET - Your Pinterest App Secret (kept secure)
  • PINTEREST_REDIRECT_URI - OAuth callback URL

Chrome Extension Integration

This backend API is designed to work with the "Pin It Your Way" Chrome extension. The extension should:

  1. Call /pinterest-auth to initiate OAuth flow
  2. Open Pinterest authorization in popup window
  3. Receive tokens via postMessage after successful auth
  4. Make API calls through /pinterest-token POST endpoint
